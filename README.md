# Obstacle Course Game (Android)

A simple Android game developed in Kotlin as part of the *User Interface Development* course.

The player controls a rocket that moves between three lanes to avoid falling asteroids.  
Each collision costs one life. After three collisions, the game resets.

## Features
- Grid-based obstacle movement (no Canvas or x/y position changes)
- Timer-driven gameplay
- Lives system with visual hearts
- Vibration and toast feedback on crash
- Space background loaded with Glide

## Technologies
- Kotlin
- Android SDK
- Glide (image loading)
